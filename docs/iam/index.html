IAM服务

IAM服务

IAM 服务是 OPSPhere 平台的系统用户和权限管理服务，主要包括如下功能：

  1. 用户和权限管理
  2. 云账号管理
  3. Cloud SSO 免密登录角色配置
  4. SSO 联邦认证 IDP 服务配置
  5. LDAP 平台登录认证配置
