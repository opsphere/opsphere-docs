费用服务

费用服务

费用服务主要用于管理客户的账单事宜，通过对所管理客户的基本信息，以及云账号信息进行配置，获取云商账单数据，并根据客户的特殊管理需求，进行折扣拆分配置，实现定制账单的生成，以及发送。 费用服务同时兼顾预警提醒功能，可对客户费用的突增突降、账号的增加减少、产品折扣的配置情况进行预警提醒。
