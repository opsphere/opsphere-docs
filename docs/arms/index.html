资源管理服务

资源管理服务

资源管理服务是一种基于项目和应用实现的云资源纳管，帮助用户将多家公有云资源按照公司内部的业务线进行关联， 主要包括如下功能：

  1. 项目和应用管理
  2. 应用与资源的关联规则管理
  3. 项目与组织架构关联
  4. 所有资源列表信息
  5. 堡垒机权限同步
